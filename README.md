
# WEBPACK + VUE2

Projeto base para criação de módulos com vuejs.


## Utilização

Ao baixar execute:

```bash
  npm install
```

Para gerar os arquivos js utilize um dos comandos:

```bash
  npm run dev
  npm run prod
  npm run dw
```
